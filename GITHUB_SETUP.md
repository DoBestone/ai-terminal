## 项目特性

在 GitHub Repository Settings 中可以启用：

- ✅ Issues - 用于用户反馈和 bug 报告
- ✅ Projects - 用于项目管理
- ✅ Wiki - 用于详细文档
- ✅ Discussions - 用于社区讨论

## 社交媒体卡片预览

当分享到社交媒体时，GitHub 会自动生成预览卡片，显示：
- 项目名称：AI Terminal

- Stars 和 Forks 数量

## Release 建议

创建第一个 Release:
1. 访问 GitHub Repository
2. 点击 "Releases" → "Create a new release"
3. Tag: `v1.0.0`
4. Release title: `AI Terminal v1.0.0 - Initial Release`
5. 描述发布内容和功能特性
